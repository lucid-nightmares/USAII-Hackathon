# Demo Script

## Demo goal
Show that SignalSafe Guardian Network is a deployable, privacy-respecting child-safety product for parents of kids 10 and under, not a generic chat classifier.

## 1. Open with the product claim
- SignalSafe is a guardian operations network for families.
- The child device scans locally.
- Only redacted incident packets reach the parent dashboard.
- Parents get immediate notification routing and a calm response workflow.

## 2. Show the top-level story
- Land on the hero section.
- Point out the product is designed for younger children.
- Call out the three pillars:
  - local-first
  - guardian response
  - deployment kits

## 3. Show that this is a family system, not a toy analyzer
- In the left rail, highlight the child fleet.
- Explain that each child can have a different install profile depending on device and age.
- Click through the install kits and explain Apple family kit, Android family agent, and school-safe tablet mode.

## 4. Run the live incident scan
- Keep Maya selected.
- Click `Scan Maya`.
- Explain that the message data shown in the demo is sample device activity.
- Emphasize that full transcript access stays local by default.

## 5. Walk through the incident queue
- Show the notification routing panel.
- Explain how push is armed first, then SMS, then backup guardian escalation.
- Show the incident cards and explain:
  - why the interaction triggered
  - what redacted evidence looks like
  - how the recommended parent actions stay focused on calm intervention

## 6. Show operational response
- Click `Acknowledge` on the top alert.
- Click `Call child` or `Escalate` on one alert.
- Scroll to the parent response log and show the action timeline updating.
- Explain that this is the key difference from a generic AI demo: the alert actually enters a guardian workflow.

## 7. Show trust boundaries
- Open `Family policy`.
- Highlight the guardian roster and trust boundary.
- Explain exactly what can leave the child device:
  - risk score
  - app and timestamp
  - redacted phrase fragments
  - escalation state
- Emphasize what does not leave by default: the raw transcript.

## 8. Close with deployment
- Open `Deployment path`.
- Show the selected install kit and deployment map.
- Explain that parents can actually imagine installing this on a younger child’s tablet or first phone.

## Closing line
SignalSafe Guardian Network uses AI to reduce harm without normalizing total child surveillance. The core value is trustworthy escalation design for families with younger kids.
